## Q :怎么关闭服务器消息转发到QQ群？

A :默认服务器消息全部转发到QQ群，如果希望关闭，服务器转发到QQ群请在`机器人正则.json`中删除：

```
    {
      "正则": "\\[Chat] <(.+)> (.+)",
      "返回消息格式": "%s1: %s2"
    },
```

也可以设置加前缀才能转发，例如要在消息前加一个`-`，可以写：

```
    {
      "正则": "\\[Chat] <(.+)> -(.+)",
      "返回消息格式": "%s1: %s2"
    },
```

## Q :我装好了机器人，结果控制台里没有任何机器人应该有的输出，而且机器人的功能也用不了怎么办？

A :请检查是否在使用`DLS.exe`开服，只有运行`DLS.exe`才能正常启动机器人
